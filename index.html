⚙️ Configurar fuente de datos

Este dashboard lee los datos directamente desde un Google Sheet publicado. Sigue estos pasos una vez:

  1. Crea o abre tu Google Sheet con el CSV de Notion.
  2. Menú Archivo → Compartir → Publicar en la web.
  3. Elige Documento completo y formato Valores separados por comas (.csv).
  4. Haz clic en Publicar y copia el enlace generado.
  5. Pégalo aquí abajo y guarda.
URL pestaña 1 — Datos de tareas (CSV)
URL pestaña 2 — Config sprints (CSV) opcional
La pestaña 2 debe tener columnas: Sprint | Inicio | Fin (ej: Sprint 3 | 02/06/2025 | 27/06/2025)
URL hoja Ayuda (AYUDA_DASHBOARD) opcional
URL imagen — Flujo de estados opcional
